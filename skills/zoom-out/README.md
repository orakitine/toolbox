# zoom-out

Pulls the agent up a layer of abstraction. Instead of drilling deeper into the code in focus, it maps the surrounding modules and callers using the project's own domain vocabulary.

## Quick start

Trigger with:

- `zoom out`
- `/zoom-out`

The next response will be a broader map of the area around the code in focus, expressed in the project's domain terms.
